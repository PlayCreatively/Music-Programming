# Parameter Playground: a playful act of exploration within a high-dimensional parameter space

## How to run:
1. Ensure you have Python installed on your system.
2. Navigate to the project directory in your terminal.
3. Run:
``` cmd
cd Project
python main.py
```

*or*

if you're running on a windows computer, you can simply double-click `run.bat` to execute the program.
